Создать пользователя

Введите email для нового пользователя и нажмите "созать"

На указанный почтовый адрес будет направлено письмо со ссылкой для регистрации

Пароль для аккаунта и название контрагента укажет пользователь при переходе на отправленную ему ссылку

Далее, назначтье пользователю доступные организации в разделе "организации". После этого данный пользователь сможет создавать фактуры.
