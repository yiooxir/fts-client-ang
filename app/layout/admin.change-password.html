Смена пароля

Смена пароля администратора

Данная страница не доступна обычному пользователю. Пароль может сменить только администратор.

Для смены пароля укажите в форме ниже старый пароль, а также новый и проверочный. Если все данные введены корректно, пароль измениться. Он будет необходим при следущем входе в программу.

Изменение пароля пользователя

Администратор также может санкционировать изменение пароля для любого пользователя.
Перейдитке к странице списка пользователей, выберите из списка строку с нужным пользователем и в разделе "управление" кликните по иконке "смена пароля для пользователя". Будет сгенерирован новый пароль для пользователя.
При этом администратор увидит новый пароль на экране, а пользователь получить уведомление с новым паролем по почте.
